# 연금계좌 납입 시뮬레이터

**얼마를 받으려면 얼마를 넣어야 할까** — 목표 월 수령액을 입력하면 필요한 납입액을 역산해 주는 단일 파일 웹 계산기입니다.

🔗 **라이브**: https://einsteindha.github.io/pension-calc/

---

## 주요 기능

| 입력 항목 | 설명 |
|---|---|
| 현재 은퇴 준비 계좌 평가액 | 이미 보유한 자산이 있을 경우 반영 |
| 현재 나이 / 수령 시작 나이 / 수령 기간 | 적립기·예치기·인출기 구간 자동 계산 |
| 증액저축 (ON/OFF) | 매년 납입액을 일정 비율씩 늘리는 방식 |
| 적립기 · 인출기 수익률 | 구간별 연 수익률 (-10% ~ +20%) |
| 목표 월 수령액 | 현재 물가 기준 입력, 물가상승 반영 역산 |
| 물가상승률 | 적립기·예치기·인출기 전 구간 반영 |

**결과 화면**
- 납입 방식 3종 비교: 매월 / 매년 말 / 매년 초 납입
- 목표 적립 원금, 총 납입액, 총 수령 예상액, 수익 배율
- 적립기 수익률별 시나리오 비교 테이블
- 현재 보유 자산의 적립 완료 시점 기여액 및 과잉 적립 경고

---

## 계산 방식

- **인출기 PV** — 물가 반영 Growing Annuity 현재가치 공식
- **납입액 역산 (PMT)** — 필요 원금 기준 연금 공식 (월납·연납 기말/기시불)
- **증액저축** — Growing Annuity FV 공식으로 첫 해 납입액 역산
- **예치 기간** — 적립 완료 후 수령 시작 전까지의 복리 운용 구간 자동 처리
- **세금·건보료·수수료 미반영** (상담용 참고 수치)

## 기술 스택

- 순수 HTML / CSS / JavaScript — 외부 라이브러리 없음
- 단일 파일(`index.html`) 배포
- 다크 모드 자동 전환 (`prefers-color-scheme`)
- 모바일 반응형 (900px / 600px 브레이크포인트)
- PWA 메타태그 (iOS 홈 화면 추가 지원)
